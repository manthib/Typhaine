Template Instantiation


FTP Instantiation

  • Open up your FTP manager and connect to your hosting.
  • Browse to your required directory (normally public_html)
  • Upload the file inside of maheer folder

Cpanel Instantiation

  • Opne your hosting site to browser.
  • Login to your hosting account.
  • Find your File Manager and open it.
  • You will See meny folders. Find public_html folder and enter it.
  • Upload the file inside of maheer folder
